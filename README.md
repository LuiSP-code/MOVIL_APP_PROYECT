# Proyecto de aplicación móvil centrada en el control de gastos

Proyecto para la materia de Programación Móvil, en donde se creo una aplicación para el control de gastos personales. Esta aplicación la desarrollamos utilizando como principales tecnologías Flutter junto al lenguaje de programación Dart para su implementación.
